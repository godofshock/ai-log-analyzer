# AI Log Analyzer & Threat Detection System

An AI-driven system that analyzes system and network logs to detect anomalous and potentially malicious behavior.

## Features

* Log parsing and feature extraction
* Anomaly detection using Isolation Forest
* Detection of suspicious patterns in system activity
* Lightweight and extensible design

## Tech Stack

* Python
* Scikit-learn

## Status

🚧 In development

## Goal

To build an intelligent system capable of identifying abnormal behavior in logs and assisting in early threat detection.

## Run

```bash
python main.py
```
